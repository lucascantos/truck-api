# truck-api

# Installation
This requires a AWS account

## Serverless

sudo apt-get install npm
npm install serverless
serverless

### Plugins

npm init
sudo npm install -g serverless-python-requirements serverless-dotenv-plugin

## Python Packages

pip -r install requirements.txt

# AWS Deploy
add aws account info
sls deploy
